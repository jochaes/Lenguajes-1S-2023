Instrucciones para ejecutar el programa: 

- En la función main del archivo "Ejercicio_3.go" puede cambiar el slice t, por cualquier otro slice de strings 
- Para modificar la dirección de la rotación y la cantidad de rotaciones
-  debe cambiar la llamada a la función "rotar" en dónde el segundo parámetro es la dirección de la rotación 
-  y el tercer parámetro es la cantidad de rotaciones 

-Ejemplo: 

rotar(&t, 0, 3) //Rotar t 3 veces a la izquierda

rotar(&t, 1, 1) //Rotar t 1 vez a la derecha

- Para ejecutar el programa 
$ go run Ejercicio_3.go
